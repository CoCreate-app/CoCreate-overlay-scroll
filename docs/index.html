CoCreate-overlay-scroll

CoCreate-overlay-scroll javascript component. Easy configuration using data-attributes and highly styleable.

Install

 
npm i @cocreate/overlay-scroll

Or you can use cdn link:

<script>https://cdn.cocreate.app/overlay-scroll/latest/CoCreate-overlay-scroll.min.js</script>

Usage

CoCreate-overlay-scroll would hide the native scrollbars, and render the overlay scrollbars by attributes, and provide the features including dragable, clickable, scrollable.

Example

							
								<div class="co_overlay-scroll dark-mode"
									data-overlay-scroll_type="scrollX;scrollY"
									data-overlay-scroll_visible="auto"
									data-overlay-scroll_pos="bottom;left"
									style="width:300px;height:300px">
									<img src="https://mdbootstrap.com/img/Photos/Others/img%20(44).jpg"/>
								</div>

Classes

  • co_overlay-scroll required

    Template main class. This class should be defined.

  • dark-mode optional

    Scrollbar theme mode. if not set, it will be light-mode as default.


Attributes

  • data-overlay-scroll_type string optional

    ScrollBar Enable/Disable. "scrollX" and "scrollY" are available. if not set, it will be enable the x-scrollbar and y-scrollbar as default.

  • data-overlay-scroll_visible string optional

    ScrollBar Behavior. "visible" and "hide", "auto" are available. if not set, it will be visable as default.

  • data-overlay-scroll_pos string optional

    ScrollBar Position. "bottom" and "top", "right", "left" are available. if not set, it will be "bottom" and "right" as default.

How Does It Works

CoCreate-overlay-scroll works based on cocreate.js and template.
